#### 开始试用

> 将项目克隆岛本地，复制~/src/componets文件夹，在main.js引入该文件夹下的index.js，通过Vue.install(ux)来安装使用。

+ 项目使用了ES6的代码，请你和babel搭配使用！

+ 项目依赖的css库：qc-ui，请你提前在你的项目中安装该UI库，并引入项目（注意CSS类名冲突），这个库也由我创建，你可以放心在你的“现代化”项目中使用！
